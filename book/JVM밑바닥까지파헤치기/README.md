## JVM 밑바닥까지 파헤치기
---

### 후기

슬슬 고연차가 되어가는 시기에 한번쯤은 JVM에 대해서 짚고 넘어가야하지 않을까? 하는 생각에 읽게 된 책.

기존에는 한글로 된 책이 없었는데 출간되었다는 소식을 듣고 반가웠지만 읽기를 미뤄두었었다.

자바, 코틀린 등 JVM 기반의 언어 개발자라면 반듯이 알고 넘어가야하는 내용들을 서술했기 때문에 몇 번을 읽어도 도움이 될 것 같은 책.

자바의 메모리 구조라던지 가비지 컬렉션, 비동기 등등에 대해 전반적으로 다루고 있으며 난이도 또한 낮지는 않다.

신입 때 읽기는 추천하지 않고 어느정도 연차가 쌓인 뒤에 이것 저것 경험이 많을 때 읽으면 좋을 것 같다.


### 좋은 점
- JVM의 중요한 요소들을 설명해주는 책으로 어려운 내용에 비해 번역이 생각보다 잘되있는 듯하다.


### 아쉬웠던 점
- 바이트 코드나 자바 코드들이 지면에 표시되는 가독성이 좋지 않았다.
