## 최고의 프롬프트 엔지니어링 강의

---

### 후기

요즘 너무나도 핫한 AI에 대해 전반적으로 배경을 알아보기 위해 읽은 책.

저자는 패스트캠퍼스에서도 강의를 냈는데 해당 강의의 초반부분이 이 책에 담겨있다.

어떻게 AI가 발전하고 있는지 프롬프트 엔지니어링이 무엇인지, 어떻게 해야하는지에 대해 전반적으로 잘 설명되어있다.

전반적인 흐름을 잡는데 이만한 책이 있을까 싶을정도로 아주 재밌게 읽었던 책.

### 좋은 점
- 전반적인 배경 설명 부분이 너무 좋았다.
- 프롬프트 엔지니어링에 대해 처음 시작하는 사람이 알기 좋게 잘 설명되어있다.

### 아쉬웠던 점
- 홍보에는 비개발자도 읽기 쉽다라고 하는데 내가 느끼기에는 비개발자가 읽기에는 난이도가 좀 있는 책이라고 생가되었다.
